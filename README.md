# apigatewaylambdaauthorizer

This project creates:
- A REST API Gateway;
    - Lambda authorizer enabled;
- A Lambda function as the authorizer of the API
    - CloudWatch Logs enabled;
- A Lambda function as the backend of the API
    - CloudWatch Logs enabled;
